# ⚡ Converter

Универсальный конвертер медиафайлов для Linux: аудио, видео и изображения — всё во всё, на базе ffmpeg. Нативный интерфейс на Qt6 (PySide6) с тёмной темой.

## Возможности

- **Пакетная конвертация** — добавьте сколько угодно файлов, они конвертируются параллельно (1–8 потоков на выбор).
- **Drag & drop** — просто перетащите файлы или целую папку в окно.
- **Управление списком** — удаляйте отдельные файлы кнопкой ✕ (в том числе прямо во время конвертации — задача отменится), очищайте весь список одной кнопкой.
- **Прогресс по каждому файлу** и общий статус внизу окна.
- **Четыре уровня качества** — без потерь / высокое / среднее / низкое (битрейт аудио, CRF видео).
- **Режим «Без потерь»** — где возможно, поток копируется без перекодирования (stream copy): извлечение аудио из видео с совпадающим кодеком, ремуксинг mp4 ↔ mkv и т. п. проходят мгновенно и байт-в-байт. Для изображений png/bmp/tiff всегда без потерь, webp — в lossless-режиме. Если копирование невозможно (кодек не помещается в целевой контейнер), используется перекодирование с максимальными настройками.
- **Выбор папки назначения** — по умолчанию рядом с исходником, либо любая папка (сброс крестиком); существующие файлы не перезаписываются — добавляется суффикс `(1)`, `(2)`…
- **Настройки запоминаются** — формат, качество, число потоков, папка назначения и размер окна восстанавливаются при следующем запуске (хранятся в `~/.config/Converter/`).

## Поддерживаемые направления

| Куда → | Форматы |
|---|---|
| Аудио | mp3, wav, flac, ogg, opus, m4a, aac |
| Видео | mp4, mkv, webm, mov, avi, gif |
| Изображения | png, jpg, webp, bmp, tiff |

На вход принимается практически всё, что понимает ffmpeg (webm, mkv, mp4, mov, avi, flv, mp3, flac, wav, opus, heic, avif и т. д.).

Особые случаи обрабатываются осмысленно:
- **видео → аудио**: извлекается звуковая дорожка;
- **аудио → видео** (например, mp3 → mp4): генерируется видеоряд с визуализацией звуковой волны;
- **видео → изображение**: берётся кадр с первой секунды;
- **видео → gif**: конвертация через палитру для качественных цветов;
- невозможные пары (например, картинка → mp3) честно помечаются как «Несовместимо».

## Требования

```bash
sudo pacman -S ffmpeg pyside6   # CachyOS / Arch
```

## Запуск

```bash
./run.sh
# или
python main.py
```

### Ярлык в меню приложений (по желанию)

```bash
# иконка приложения
mkdir -p ~/.local/share/icons/hicolor/scalable/apps
cp converter/ui/assets/icon.svg ~/.local/share/icons/hicolor/scalable/apps/converter.svg
# сам ярлык (поправь Exec/Path внутри, если проект лежит не в ~/Code Projects/Converter)
cp converter.desktop ~/.local/share/applications/
```

## Структура проекта

```
main.py                    — точка входа
converter/
  formats.py               — форматы и построение аргументов ffmpeg
  probe.py                 — чтение метаданных через ffprobe
  jobs.py                  — очередь задач, запуск ffmpeg, прогресс
  ui/
    main_window.py         — главное окно
    file_row.py            — карточка файла в списке
    theme.py               — тёмная тема (QSS)
    assets/                — иконка приложения и элементы интерфейса (SVG)
```
